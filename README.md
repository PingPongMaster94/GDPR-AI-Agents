# GDPR-AI-Agents

This repository contains both the implementation code and the academic literature used for the **Systematic Literature Review (SLR)** phase of my thesis on GDPR compliance and AI agents.

---

# Repository Structure

## Code/

Contains the implementation and experimentation components of the project, including:

- Source code
- Data processing pipelines
- Models and experiments
- Notebooks
- Annotated datasets
- Supporting scripts and utilities

### Main Subfolders

- `src/` → Core source code
- `models/` → Model-related files
- `notebooks/` → Research and experimentation notebooks
- `data/`
  - `annotated/`
  - `processed/`
  - `raw_law/`
  - `raw_policies/`

---

## SLR/

Contains all academic papers and materials used during the Systematic Literature Review process.

### Structure

#### `1. Contextual/`
Background literature providing foundational concepts and broader context related to:
- GDPR
- AI governance
- Compliance systems
- Data protection regulation

#### `2. Supporting/`
Studies indirectly supporting the research topic, including:
- NLP for compliance
- Privacy policy analysis
- Legal automation
- AI-assisted governance frameworks

#### `3. Core/`
Key papers directly related to the central thesis research questions and implementation focus.

---

# Purpose

The goal of this repository is to:

- Organize thesis-related code and research materials systematically
- Maintain traceability between implementation and academic sources
- Support reproducibility of experiments and literature review processes
- Centralize all thesis artifacts in a single repository

---

# Notes

- Research papers are indexed numerically for easier citation and tracking
- Literature is categorized by relevance to the thesis objectives
- The repository structure may evolve throughout the thesis development lifecycle
- `.DS_Store` and other system-generated files are excluded from version control

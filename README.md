# Thesis Studies Repository

This repository contains the academic papers and materials used for the **Systematic Literature Review (SLR)** phase of my thesis.

## Structure

* **1. Contextual**
  Background literature that provides general context and foundational concepts.

* **2. Supporting**
  Relevant studies that contribute indirectly to the research topic.

* **3. Core**
  Key papers that directly address the main research questions.

* **Low Relevance**
  Papers reviewed but deemed less relevant to the final analysis.

---

## Purpose

The goal of this repository is to:

* Organize research papers systematically
* Maintain traceability of sources used in the thesis
* Support reproducibility of the literature review process

---

## Notes

* File names are indexed for easier reference during analysis
* Papers are categorized based on relevance to the research questions
* This structure may evolve as the review progresses
